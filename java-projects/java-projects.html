Tic-Tac-Toe

This project is a terminal-based version of the classic Tic-Tac-Toe game, designed for quick and easy command-line play. It detects winning combinations (horizontal, vertical, diagonal) or declares a tie when no spaces are left. The game provides clear prompts and updates after each turn.

Key Prompts:


  • Choose your opponent: play against a robot or a friend.

  • Select your symbol: X or O for the first player.

Game Updates:


  • Display available spaces on the board with corresponding numbers.

  • Announce the winner or declare a tie.

  • Prompt to restart the game after it ends.
