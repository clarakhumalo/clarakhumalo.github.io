Tic-Tac-Toe

This project is a terminal-based version of the classic Tic-Tac-Toe game, designed for quick and straightforward gameplay via the command line. The game checks for winning combinations (horizontal, vertical, diagonal) or declares a tie if no spaces remain. The interface provides clear prompts and updates after each turn.

Key Features:


  • Option to play against a friend (human) or a robot.

  • First player can choose their symbol (X or O).

  • Displays available spaces on the board with corresponding numbers.

  • Announces the winner or declares a tie.

  • Prompts to restart the game after it ends.
